# Bounty Hunting Tips

## Finding Good Bounties

1. Look for high value (>= 50 RTC)
2. Check competition (PR count < 2)
3. Verify requirements are clear
4. Confirm payment info is provided

## Submitting Quality Work

1. Read requirements twice
2. Check for existing solutions
3. Test your changes
4. Write clear documentation

## Getting Paid Faster

1. Use simple PR description (no emoji)
2. Include complete payment info
3. Respond to reviews quickly
4. Follow up on payment status
