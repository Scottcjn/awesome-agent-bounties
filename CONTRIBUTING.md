# Contributing to awesome-agent-bounties

Thank you for your interest in contributing! This list tracks bounty programs for AI agents and autonomous contributors.

## How to Contribute

### Adding a New Entry

1. **Fork the repository** and clone it locally
2. **Add your entry** to the appropriate section in `README.md`
3. **Follow the format**: `- [Project Name](link) - Brief description`
4. **Verify relevance**: Only add projects with AI agent bounties or autonomous contribution programs
5. **Submit a Pull Request** with a clear description

### What Qualifies

- Projects with AI agent bounty programs
- Platforms that accept autonomous contributions
- Tools and frameworks for agent-based development
- Documentation of agent bounty mechanisms

### Guidelines

- **Agent-focused**: Entries should relate to AI agents or autonomous systems
- **Active programs**: Prefer projects with active bounty programs
- **Clear descriptions**: Mention payment type and agent requirements
- **Alphabetical order**: Keep lists sorted within sections
- **Valid links**: Test all URLs before submitting

### Pull Request Checklist

- [ ] Entry follows the existing format
- [ ] Related to AI agents or autonomous contributions
- [ ] Description mentions bounty/payment details
- [ ] Added to the correct section
- [ ] No duplicate entries

## Code of Conduct

- Be respectful and inclusive
- Focus on constructive feedback
- Support the AI agent community

## Questions?

Open an issue for discussions or clarifications.
