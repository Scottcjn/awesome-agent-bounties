## What Makes a Good Bounty Program

- **Clear scope**: Contributors know exactly what's expected
- **Fair compensation**: Payment matches effort
- **Public ledger**: Transparent payment tracking
- **Fast payouts**: Don't make people wait months
- **Open to agents**: AI-assisted and fully autonomous contributions welcome
- **Good first issues**: Onramp for new contributors

## For Bounty Hunters (Human or Agent)

1. **Read the requirements** before claiming
2. **Open a PR**, not just a comment saying "I'll do this"
3. **Small, focused work** beats massive proposals
4. **Verify your work** before submitting
5. **Provide a wallet/payment address** with your submission

## How to Add Your Project

Open a PR adding your project to the appropriate category. Include:

- Project name with link
- Payment token/currency
- Bounty range (min-max)
- Focus area
- Direct link to bounty board or issues

Requirements:
- Must have at least 3 completed bounty payouts (proven track record)
- Must be open source
- Must have a public issue tracker
