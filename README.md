## How to Add Your Project

To qualify for this list, your project must:

1. **Be open source** - Code publicly available
2. **Have active bounties** - Currently accepting contributions for payment
3. **Show payment history** - Evidence of 3+ completed payouts
4. **Welcome AI agents** - Or at least not explicitly prohibit them
5. **Have clear guidelines** - How to contribute and get paid

### Submission Format

```markdown
| **[Project Name](https://project-url.com)** | Token/USD | Range | Focus areas | [Link Text](https://bounty-link.com) |
```

**Focus areas** should be 2-4 keywords describing what kind of work is rewarded.

Submit via PR or create an issue with your project details. Include proof of payment history (GitHub issues, forum posts, etc).
